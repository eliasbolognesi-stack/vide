# 🤖 Etapa 3 - CRIANDO OS PROMPTS

Os prompts são as instruções que você dará à IA para gerar imagens e vídeos. Quanto mais detalhado, melhor o resultado!

---

## 📸 Fórmula de um Bom Prompt

```
[OBJETO] + [AÇÃO/CONTEXTO] + [ESTILO] + [QUALIDADE]
```

### Exemplo Completo:
"Estádio de futebol moderno durante a Copa do Mundo, milhares de torcedores em pé comemorando, luzes cinematográficas azuis e amarelas, detalhes ultra realistas, qualidade 8K"

## 💡 Dicas para Prompts Melhores

✅ **Use detalhes específicos:**
```
❌ "Estádio"
✅ "Estádio de futebol moderno, noite, luzes LED brancas e azuis, 80.000 torcedores"
```

✅ **Especifique o estilo:**
```
❌ "Copa do Mundo"
✅ "Copa do Mundo, estilo cinematográfico, iluminação épica, cores vibrantes, 4K"
```

✅ **Defina a qualidade:**
```
Adicione sempre: "ultra realista", "4K/8K", "qualidade profissional", "fotografia profissional"
```

✅ **Descreva emoções e ações:**
```
❌ "Pessoas comemorando"
✅ "Multidão em êxtase abraçando-se, pulos de alegria, confetes caindo, risos"
```
