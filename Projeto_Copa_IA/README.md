# 🎬 Projeto Copa do Mundo 2026 - Vídeo com IA

Bem-vindo ao seu projeto de produção audiovisual! Este é um guia completo para criar um vídeo criativo sobre a Copa do Mundo utilizando ferramentas de Inteligência Artificial.

## 📋 Resumo do Projeto

- **Tema:** Copa do Mundo 2026
- **Objetivo:** Criar um vídeo de 45-60 segundos com elementos visuais gerados por IA
- **Público-alvo:** Fãs de futebol e entusiastas de tecnologia

## 🎯 Componentes Obrigatórios

✅ Imagens geradas por IA  
✅ Narração por IA ou gravada  
✅ Música de fundo (sem direitos autorais)  
✅ Transições e efeitos visuais  
✅ Tela final com créditos  

## 📁 Estrutura de Pastas

```
Projeto_Copa_IA/
├── 01_Planejamento/          → Responda as perguntas iniciais
├── 02_Roteiro/               → Estruture as cenas
├── 03_Prompts/               → Prompts para IA
├── 04_Imagens/               → Imagens geradas
├── 05_Videos/                → Vídeos curtos gerados
├── 06_Naracao/               → Arquivos de áudio
├── 07_Musica/                → Fundo musical
├── 08_Montagem/              → Projeto em Canva/CapCut
├── 09_Creditos/              → Design da tela final
└── 10_Final/                 → Vídeo MP4 final
```

## 🚀 Como Começar

1. **Abra** `01_Planejamento/PLANEJAMENTO.md`
2. **Responda** as 3 perguntas iniciais
3. **Siga** cada etapa sequencialmente
4. **Use** os exemplos como inspiração

## 💡 Dicas Importantes

### Composição Visual
- Regra dos terços (não coloque tudo no centro)
- Distribua elementos pela tela
- Cores relacionadas ao futebol: 🟢 Verde, 🟡 Amarelo, 🔵 Azul, ⚪ Branco

### Timing
- Cada cena: 4-7 segundos
- Narração clara e envolvente
- Efeitos sonoros discretos

### Qualidade
- Imagens em alta resolução (mínimo 1080p)
- Transições suaves e curtas (0.5-1s)
- Áudio balanceado entre música e narração
