# 🧩 SLIDES PRONTOS PARA COPIAR (base do vídeo)

Ficha de construção da apresentação base (16:9 = 1920×1080). Cada slide é uma cena.
**Como usar:** no Canva, crie um design **Apresentação (16:9)** → adicione 9 páginas → em cada página, copie o texto abaixo, aplique as cores e arraste a imagem/clipe da cena.

> 💡 Para exportar como vídeo no Canva: **Compartilhar → Baixar → MP4 Video**. Ajuste o tempo de cada página em **Cronometrar** (ícone de relógio) e adicione transições.

---

## 🎨 Paleta de cores (use estes códigos exatos)

| Cor | Código (HEX) | Uso |
|---|---|---|
| 🟢 Verde gramado | `#009C3B` | Destaques, fundo da mensagem final |
| 🟡 Amarelo | `#FFDF00` | Palavras de destaque, títulos |
| 🔵 Azul | `#002776` | Detalhes, fundo alternativo |
| ⚪ Branco | `#FFFFFF` | Texto principal |
| ⚫ Fundo escuro | `#0B1A12` | Fundo das cenas com imagem |

**Fontes sugeridas:** Título → *Bebas Neue* ou *Anton*; Texto → *Montserrat Bold*.
**Regra de ouro:** texto sempre com sombra/contorno para ler sobre qualquer imagem; nada centralizado demais (regra dos terços).
