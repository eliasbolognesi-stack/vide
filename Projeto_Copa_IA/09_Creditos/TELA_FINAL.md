## 🎨 Como deixar a tela bonita

- **Fundo:** verde-amarelo (degradê) ou uma imagem desfocada da taça
- **Cores do texto:** branco com leve sombra (alto contraste)
- **Fonte:** a mesma usada no resto do vídeo (consistência)
- **Layout (regra dos terços):** título em cima, integrantes no meio, ferramentas embaixo
- **Toque final:** um pequeno logo (veja desafio extra) no canto

### Esboço de layout
```
┌─────────────────────────────────────┐
│        🏆 BRASIL 2026 com IA          │   ← título (topo)
│                                       │
│   Integrantes: Nome 1 · Nome 2 · ...  │   ← centro
│                                       │
│   Ferramentas: Firefly · ElevenLabs   │   ← rodapé
│                · CapCut · Canva        │
└─────────────────────────────────────┘
```

---

## 🛠️ Como criar no Canva / CapCut

**Canva:**
1. Adicione uma nova página/cena de 5 s ao final
2. Escolha um fundo (cor sólida, degradê ou imagem desfocada)
3. Adicione caixas de texto com o conteúdo acima
4. Anime o texto (Fade / Subir) para entrada suave

**CapCut:**
1. Adicione um clipe de cor ou imagem no fim da linha do tempo (5 s)
2. Use **Texto** para cada bloco (Projeto / Integrantes / Ferramentas)
3. Aplique animação de entrada e um **Fade out** no final do vídeo
