# 🎨 Etapa 4 - GERANDO AS IMAGENS

Agora é hora de usar os prompts e gerar as imagens com IA!

---

## 📸 Passo a Passo para Gerar Imagens

### Opção 1: Microsoft Designer

1. Acesse: https://designer.microsoft.com
2. Clique em "Criar"
3. Cole o prompt da Etapa 3
4. Clique "Gerar"
5. Espere 30-60 segundos
6. Faça download da melhor imagem
7. Salve na pasta `04_Imagens/` com nome descritivo

### Opção 2: Canva

1. Acesse: https://www.canva.com
2. Busque "AI Image Generator"
3. Digite seu prompt
4. Clique "Generate"
5. Escolha a melhor versão
6. Download em alta resolução
7. Salve na pasta `04_Imagens/`

### Opção 3: Adobe Firefly

1. Acesse: https://firefly.adobe.com
2. Cole seu prompt
3. Ajuste parâmetros (cores, estilo)
4. Clique "Generate"
5. Download a imagem
6. Salve na pasta `04_Imagens/`

## ⚙️ Dicas para Melhorar Resultados

### Se a imagem não saiu como esperado:

1. **Reescreva o prompt com mais detalhes**
   - Adicione cores específicas
   - Descreva posições dos objetos
   - Especifique ângulo da câmera

2. **Ajuste parâmetros**
   - Aumentar resolução pedida
   - Escolher estilo (fotografia, pintura, cinema)
   - Usar "negative prompts" (o que NÃO quer ver)

3. **Gere várias versões**
   - A IA oferece múltiplas versões
   - Escolha a melhor ou remixe com outras

### Exemplo de Melhoria:

❌ Versão 1 (genérica):
"Estádio"

✅ Versão 2 (melhorada):
"Estádio moderno noturno, luzes LED brancas e azuis, 80 mil torcedores em pé, 
bandeiras do Brasil, confetes no ar, movimento de câmera suave, cinematográfico"

## 🎬 Redimensionamento das Imagens

Para usar em vídeo, todas as imagens devem ter o mesmo tamanho:

**Resolução recomendada:** 1920x1080 pixels (Full HD)

Como redimensionar:
- Online: https://www.pixlr.com/resize/ ou https://www.canva.com/resize/
- Windows: Paint ou Photos
- Ferramentas: Canva, Photoshop, GIMP
