# 🎬 Etapa 5 - CRIANDO VÍDEOS COM IA

Transforme suas imagens e prompts em vídeos animados!

---

## 📹 Ferramentas para Gerar Vídeos

### Opção 1: Canva Video Editor
1. Acesse: https://www.canva.com
2. Procure por "AI Video"
3. Cole o prompt
4. Gere vídeo (5-7 segundos)
5. Download em MP4
6. Salve na pasta `05_Videos/`

### Opção 2: CapCut (Recomendado)
1. Baixe: https://www.capcut.com
2. Crie novo projeto
3. Use "Generate Video" ou "Text to Video"
4. Cole o prompt
5. Configure duração: 5-7 segundos
6. Gere e exporte
7. Salve na pasta `05_Videos/`

### Opção 3: Pika Labs
1. Acesse: https://pika.art
2. Faça login
3. Clique "Generate"
4. Cole o prompt
5. Selecione duração
6. Gere o vídeo
7. Download em MP4

## ⚡ Dicas para Vídeos Melhores

### 1. Especifique Movimento
```
❌ "Estádio durante Copa do Mundo"
✅ "Câmera se movimentando suavemente pela lateral do estádio, 
   foco nos torcedores emocionados"
```

### 2. Defina Duração Exata
```
"Vídeo de 6 segundos mostrando..."
```

### 3. Use Slow Motion
```
"Vídeo em slow motion (câmera lenta) de jogador chutando"
```

### 4. Descreva Transições
```
"Transição suave do estádio para torcedores celebrando"
```

### 5. Qualidade Visual
```
Sempre adicione: "cinematográfico", "4K", "ultra realista", "iluminação profissional"
```

---

## 🔧 Se o Vídeo não Ficar Bom

1. **Refaça com prompt mais detalhado**
   - Adicione mais descrição visual
   - Especifique cores e iluminação
   - Defina movimento da câmera

2. **Ajuste a duração**
   - Às vezes vídeos curtos ficam melhores
   - Tente 4 segundos vs 7 segundos

3. **Tente outra ferramenta**
   - CapCut é melhor para slow motion
   - Canva é bom para gifs curtos
   - Pika Labs é mais realista

4. **Gere várias versões**
   - Crie 3-5 versões de cada vídeo
   - Escolha a melhor
